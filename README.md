# Currículo - Sofia Figueiredo

## 📄 Sobre
Currículo online desenvolvido em HTML e CSS como parte do meu portfólio de desenvolvedora Full Stack.

## 🎨 Características
- Design moderno e responsivo
- Layout em duas colunas (inspirado em currículos profissionais)
- Animações suaves com CSS
- Interatividade com JavaScript
- Otimizado para impressão (print-friendly)

## 🛠️ Tecnologias
- HTML5
- CSS3 (Grid, Flexbox, Variáveis CSS)
- JavaScript (interações básicas)
- Font Awesome (ícones)
- Google Fonts (Poppins, Roboto Mono)

## 🚀 Como visualizar
1. Clone o repositório
2. Abra o ficheiro `index.html` no navegador
3. Ou aceda à versão online: [GitHub Pages](https://sofiadcfigueiredo.github.io/curriculo)

## 📱 Responsividade
- Desktop: layout em duas colunas
- Tablet/Mobile: layout em uma coluna
- Impressão: otimizado para PDF
